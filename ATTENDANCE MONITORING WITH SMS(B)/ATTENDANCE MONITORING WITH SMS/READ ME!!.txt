DATABASE SETUP

1.Download the zip file "LUBANG INTEGRATED SCHOOL" and extract.
2.Download and install XAMPP
3.Run the XAMPP control panel and start MySQL and Apache
4.Open the browser and go to http://localhost/phpmyadmin/ to create the database
5.Click the new to create a database.
6.Name the database "lubang_db"
7.Click import to import the sql file.
8.Click choose file and select the file that can be found inside the "LUBANG INTEGRATED SCHOOL" folder
